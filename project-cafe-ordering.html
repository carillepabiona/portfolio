Project Overview

Problem · Solution · Outcome

Problem

Cafes faced slow table-side ordering, manual paper menus, and disconnected POS and kitchen workflows — resulting in long customer wait times, order mistakes, and no real-time visibility into kitchen operations.

Solution

Developed a web-based system using ASP.NET Core where customers scan a QR code on their table to place orders, which are immediately pushed to both the POS and KDS for seamless processing and fulfillment.

Outcome

Created a fully connected cafe ecosystem — from customer QR ordering to kitchen fulfillment — eliminating manual steps and improving accuracy and speed. Delivered as a polished academic project showcasing web development expertise.

Development Timeline

Planning & Research

Analyzed real-world cafe workflows to identify pain points and define system requirements for QR ordering, POS, and KDS.

UI/UX Design

Designed all page layouts for customer-facing QR menu, cashier POS, kitchen KDS, and admin management pages using Bootstrap.

Database Design

Built SQL Server schema covering orders, items, tables, users, and QR code assignments. Configured EF Core models and migrations.

Development

Implemented all modules with ASP.NET Core Razor Pages, wired up EF Core for data access, and built the QR code generation and order routing logic.

Testing

End-to-end testing of the QR → POS → KDS order flow, validated sync accuracy, and fixed edge cases in order status transitions.

Deployment & Demo

Deployed locally for academic presentation with a full live demonstration of the QR-to-kitchen workflow.

Learnings

Challenges & Solutions

Challenge

Keeping POS and KDS in sync after QR order submission without page refresh.

Solution

Used JavaScript polling with AJAX to periodically fetch new orders from the server and update the POS and KDS displays automatically.

Challenge

Generating unique QR codes per table and routing orders correctly.

Solution

Generated table-specific URL tokens encoded as QR codes. Each token maps to a table record in SQL Server, ensuring orders are tagged with the correct table ID.
